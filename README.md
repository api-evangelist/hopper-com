# hopper-com

Hopper — price prediction consumer travel app and Hopper Technology Solutions (HTS) B2B travel fintech.

## APIs

- **HTS Airline API v1.1** — OAuth2-secured REST API for embedding Cancel For Any Reason (CFAR) and Disruption Guarantee (DG) ancillaries in airline booking flows. 23 operations under `https://airlines-api.hopper.com/airline/v1.1`. Spec: [openapi/hopper-airlines-openapi.yml](openapi/hopper-airlines-openapi.yml).

## SDKs

- [hc-airlines-java](https://github.com/hopper/hc-airlines-java) — Java client (OpenAPI-generated).
- [hts-airlines-dotnet](https://github.com/hopper/hts-airlines-dotnet) — .NET client (OpenAPI-generated).
- [hc-airlines-angular](https://github.com/hopper/hc-airlines-angular) — Angular client library.
- [hc-hotels-ios-sdk](https://github.com/hopper/hc-hotels-ios-sdk) — iOS Price Freeze SDK.
- [hc-hotels-demo](https://github.com/hopper/hc-hotels-demo) — Web SDK demo for hotel Price Freeze.
